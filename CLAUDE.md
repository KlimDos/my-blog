## Project Overview

A Hugo-based static blog for motorcycle news, published at `blog.alimov.top`. Content is in Russian, using the Mainroad theme. Deployed via GitHub Actions to GitHub Pages.

## Commands

```bash
# Local development server
hugo server -D

# Production build
hugo --gc --minify

# Initialize/update theme submodules
git submodule update --init --recursive
```

Hugo extended version is required (see `.github/workflows/hugo-cd.yaml`).

## Architecture

**Config:** `hugo.toml` — base URL, theme, pagination, taxonomies, permalinks (`/:contentbasename/`), and Goldmark markdown settings.

**Content:** `content/posts/YYYY/MM/post-name.md` with YAML frontmatter (`title`, `date`, `categories`, `tags`, `cover` image URL, `source` URL).

**Layouts:** Custom templates in `layouts/` override theme defaults. `layouts/_default/baseof.html` is the root HTML shell. Partials in `layouts/partials/` extend the Mainroad theme.

**Theme:** `themes/mainroad/` (active), plus `themes/PaperMod/` and `themes/stack/` as unused git submodules.

**Static assets:** `static/css/extra.css` has custom CSS (accent color: `#ff5722`). `static/CNAME` sets the custom domain.

**i18n:** `i18n/ru.yaml` overrides theme strings for Russian.

**Output:** `public/` (gitignored) — generated by Hugo, deployed by CI.

## Deployment

Push to `main` triggers `.github/workflows/hugo-cd.yaml`, which builds with `hugo --gc --minify` and deploys the `public/` directory to GitHub Pages.
